LED Control Panel

Apply options if needed. Choose one of the ways to adjust the color of the LED strip. Hex values have the prefix "0x" or "#".

Options

Color Correction

Enter correction values for each color channel (0-255).






Solid Color

Enter a single color value (integer or hex) to set all LEDs to the same color.




Gradient

Enter number of LEDs per segment, and provide 3 gradient values (integer or hex).







Individual Pixel Color Control

Enter comma-separated values (integer or hex, e.g. 0x12FA8C). One value per LED.
